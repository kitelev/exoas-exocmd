# Resolve archive ontology

NamedQuery for the «Archive Ontologically» command (RFC 78c2b7d0 C5). Computes
the archive-target ontology of `$currentAsset` by a 1-hop navigation through its
source ontology's `exo__Ontology_archiveOntology` declaration. Consumed read-side
by the `property_set` grounding's `targetValueQuery` (C4 CQRS bridge); `$currentAsset`
is auto-injected with the click/apply target IRI.

```sparql
SELECT ?archiveOnto WHERE { $currentAsset <https://exocortex.my/ontology/exo#Asset_isDefinedBy> ?o . ?o <https://exocortex.my/ontology/exo#Ontology_archiveOntology> ?archiveOnto }
```
