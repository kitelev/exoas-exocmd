# exocmd__CommandBinding_targetPrototype

Prototype-scoped command binding target. Declared retroactively: the resolver
has read this predicate since #3272-era work (`CommandBindingProperty.TARGET_PROTOTYPE`),
and since PR #3804 (v16.170.0, req `5ad0d6b4`) it matches the target's
**transitive** `exo__Asset_prototype` chain, so one binding on a parent
prototype covers instances of all (transitive) child prototypes.
