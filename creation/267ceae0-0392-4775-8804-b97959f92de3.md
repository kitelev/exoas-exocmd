# Create Next Iteration

Кнопка для создания следующей итерации текущей задачи. Use case driver: [[d8725272-4b4d-4e68-bf01-ce9b111bbe12|Sub-1min UI extension]].

- **Grounding**: [[d240c457-3d62-4813-99c4-922134789771|Create Next Iteration grounding]] (`create_instance` с `linkBackProperty=ems__Effort_prevIteration`)
- **Icon**: plus-square (Lucide)
- **Category**: creation
- **Effect**: создаётся новый `ems__Task` с copied frontmatter (см. blacklist в RFC) + `ems__Effort_prevIteration` → текущая задача
