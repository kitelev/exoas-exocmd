# Allow transition to ToDo (from Analysis / Backlog)

SPARQL ASK precondition: returns `true` only when current status ∈ {Analysis, Backlog}.

State-machine forward transition: standard workflow is Backlog → Analysis → ToDo, but small-scope assets may skip Analysis and go Backlog → ToDo directly (Project semantics from v15.38.0).

Used by "Move to ToDo" command.
