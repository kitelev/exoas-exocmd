# Create Class → exo__Ontology binding

`exocmd__CommandBinding` (project 85150d63, W3). Binds the «Create Class»
command to the `exo__Ontology` **metaclass** (UID
`829b9b3b-6fc3-4276-be6a-27d3398c012e`), so the inline button renders on every
ontology-definition page (every asset whose `exo__Instance_class` includes
`exo__Ontology`). No precondition — always available on ontology pages.

`targetClass` is the UID form (not the `exo__Ontology` label) so the match is
independent of metadata-cache label resolution — the raw UID is always present
in the host's expanded class set (`extractAssetClasses`, Issue #3141).
