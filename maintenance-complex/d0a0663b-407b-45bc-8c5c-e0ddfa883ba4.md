# Rename to UID

Renames the target asset's file to match its `exo__Asset_uid` value, updating all wikilinks across the vault.

- **Precondition**: Visible only when filename does not match UID (host function: `hasNonUidFilename`)
- **Grounding**: Calls `renameToUid` service
- **Icon**: file-digit (Lucide)
- **Category**: maintenance
