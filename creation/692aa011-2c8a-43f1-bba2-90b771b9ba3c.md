# Create fleeting note

Dynamic command that creates a new `ztlk__FleetingNote` asset.

- **Grounding**: Calls `createAsset` service with `ztlk__FleetingNotePrototype`
- **Icon**: sticky-note (Lucide)
- **Category**: creation
- **Input**: Note title (text field)
- **Precondition**: None (always available)
- **Palette**: registered in Obsidian Command Palette via `ExocmdCommandPaletteRegistrar` (RFC `1429fcd0`). `paletteId: "create-fleeting-note"` is the stable id for user hotkey bindings — must NOT change without a migration. Previously implemented as the TypeScript `CreateFleetingNoteCommand` class in `packages/obsidian-plugin`; the TS code was deleted in PR-3 of the same RFC.
