# Allow transition to Doing (from ToDo / Backlog)

SPARQL ASK precondition: returns `true` only when current status ∈ {ToDo, Backlog}.

State-machine forward transition mirrors v15.38.0 `canStartEffort`: Tasks/Meetings start from Backlog, Projects from ToDo. The grounding is composite (status + startTimestamp).

Used by "Set Status Doing" command.
