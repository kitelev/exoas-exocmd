# Set Status Cancelled

Sets `ems__Effort_status` to Cancelled AND stamps `ems__Effort_resolutionTimestamp` with the current local timestamp on the target asset.

- **Precondition**: Visible only when asset is NOT in Cancelled status
- **Grounding**: Composite (`d612e5f5`) — sets status + resolutionTimestamp atomically (mirrors v15.38.0 `trash-effort` behaviour)
- **Icon**: x-circle (Lucide)
- **Category**: status
