# $nowTimestamp

`exocmd__SubstitutionToken` instance. Resolves at execution time to local ISO 8601 timestamp without timezone suffix (e.g. `2026-05-25T01:05:00`). Parameterless. Matches `DateFormatter.toLocalTimestamp(clock.now())` shape used by all other creation services (Issue #3188).

## Usage

Referenced from `exocmd__PropertyDefault_value` for createdAt / updatedAt fields. RFC 727572d2 Universal Default uses this for `exo__Asset_createdAt` of every newly created instance.
