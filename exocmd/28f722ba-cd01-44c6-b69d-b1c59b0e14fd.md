# Not archived

Reusable **AtomicPrecondition** — returns `true` when the target is **not**
archived. The archived-guard for the mass archived-gate (req(ems)
`f1fbea08-f08f-4605-9c41-bf20c90b395f`).

`ASK { FILTER NOT EXISTS { $target exo:Asset_archived "true" } }` — the exact
inverse of the «Visible only when the task is archived» atomic
(`008c6d16`) built for «Un-archive». The bare `archived` frontmatter key is
whitelisted → the `exo:Asset_archived` triple with literal value `"true"` (RFC
871c1e56).

Added as one conjunct of a per-command `AllPrecondition` wrapper on every task
command that would otherwise remain visible on an **archived** task (Done +
`archived "true"`). Reused across the archived-gated commands (Create Action,
Create Related Task, Convert to Project, Re-open, Clean Properties, Copy Label to
Aliases, Repair Folder, Fix Missing Label, Rename to UID) — making the guard
non-vacuous (a leaf in ≥2 composites). «Un-archive» is deliberately NOT gated
(it must show precisely when archived).
