# Clean Properties

Removes **empty** frontmatter properties from the target asset — values that are
`null`, an empty string, an empty array (`[]`), or an empty object (`{}`).

- **Precondition**: Visible only when the asset has at least one empty property
  (host function: `hasObsoleteProperties` — a legacy identifier; the check it
  computes is "has empty frontmatter values" via the core `hasEmptyProperties`
  helper).
- **Grounding**: Calls the `cleanProperties` service
  (`PropertyCleanupService.cleanEmptyProperties`).
- **Icon**: eraser (Lucide)
- **Category**: maintenance
