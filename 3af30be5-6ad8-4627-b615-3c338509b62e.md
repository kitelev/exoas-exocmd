# workflow_transition forward

Generic dispatcher Grounding for forward (`isRollback=false`) workflow transitions. Resolves transition at runtime from `(assetClass, currentStatus)` against vault Workflow ABox via `GroundingExecutor.executeWorkflowTransition` (RFC 36347daf Phase 2).

Available for future Phase 3 migration of forward Commands (`Move to Backlog`, `Move to Analysis`, `Move to ToDo`, `Mark as done`, etc.). Not yet referenced by any Command — created in Phase 3 alongside the rollback Grounding for symmetry.
