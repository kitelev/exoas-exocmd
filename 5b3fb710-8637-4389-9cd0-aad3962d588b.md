# exocmd__InheritanceRule_targetClassExclusion

`exo__ObjectProperty` on `exocmd__InheritanceRule`. Multi-valued list wikilinks на `exo__Class`. Cardinality 0..N.

YAML serialization:
```yaml
exocmd__InheritanceRule_targetClassExclusion:
  - "[[<class-uuid-1>]]"
  - "[[<class-uuid-2>]]"
```

Engine filter: drop rule if `targetIRI rdf:type ?cls AND ?cls ∈ targetClassExclusion`.

Resolves the «Area-rule applies → suppress generic parent-rule» case (RFC v2 §Engine semantics step 2) — generic rule's targetClassExclusion contains `ems__Area`, so it does NOT fire when target is an Area.
