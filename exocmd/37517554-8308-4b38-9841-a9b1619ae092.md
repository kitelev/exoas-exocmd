# Visible until Done (atomic action)

SPARQL ASK precondition: returns `true` while the target's `ems__Effort_status` is anything **except** Done. Unlike the Task state-machine precondition (`577f798f`, Doing→Done only), this permits a one-tap **any-status → Done** transition for atomic `ems__Action` assets (e.g. medication intake: Backlog → Done in a single tap). Hides the button once the action is Done.
