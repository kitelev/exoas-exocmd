# Trash → Action binding

Binds the generic "Trash" command (`0c3debe0`) to `ems__Action` assets (epic
`40188c71` Trash button). The Trash command sets `ems__Effort_status` → Trashed
(`5d14f18d`) + `ems__Effort_resolutionTimestamp` → now + a deliberate-drop
`ems__Effort_trashedReason`, via its composite grounding `d612e5f5` — matching the
vision (status Trashed + resolvedAt). Precondition `36ae0a02` hides it once the
action is already Trashed. Reuses the same command already bound to Task/Project.
