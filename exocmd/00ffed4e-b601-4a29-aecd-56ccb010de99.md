# Not in Today zone

SPARQL ASK precondition: returns `true` when task zone is NOT Today AND task is not in a terminal status (Done / Cancelled / Trashed / Failed).

**IRI forms** — zone targets use the *filename-IRI* form because the zone enum assets in `shared-identities` carry a human-language label (`Зона критичности…`) which doesn't parse as `<prefix>__<LocalName>`, so the parser emits `obsidian://vault/.../<uuid>.md`. Status enums in `assetspaces/ems` are labelled `ems__EffortStatusDone` etc., so the parser emits the symbolic IRI `https://exocortex.my/ontology/ems#EffortStatusDone`.

Used by "Set Criticality High" command to show the button only when relevant.
