# Has non-UID filename

Host function precondition: returns `true` when the target asset's filename does not match its `exo__Asset_uid` (i.e., is a human-readable name instead of UUID-based).

Evaluated via `PreconditionEvaluator.registerHostFunction("hasNonUidFilename", ...)`.
Uses `EvalContext.fileBasename` to compare against the asset's UID.
Used by the **Rename to UID** command.
