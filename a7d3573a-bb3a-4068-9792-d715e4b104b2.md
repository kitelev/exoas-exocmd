# Remove Start Timestamp

Dynamic command that removes `ems__Effort_startTimestamp` from an asset.

- **Precondition**: Visible only when the asset has a start timestamp
- **Grounding**: Deletes the `ems__Effort_startTimestamp` property
- **Icon**: clock-x (Lucide)
- **Category**: maintenance

This is the reference example from RFC-009 Section 6.
