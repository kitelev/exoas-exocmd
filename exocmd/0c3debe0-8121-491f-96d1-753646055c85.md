# Trash

Sets `ems__Effort_status` to Trashed, stamps `ems__Effort_trashedReason` («Неактуально / осознанный отказ» — deliberate-drop default) AND `ems__Effort_resolutionTimestamp` with the current local timestamp on the target asset.

- **Precondition**: Visible only when asset is NOT already Trashed
- **Grounding**: Composite (`d612e5f5`) — sets status + reason + resolutionTimestamp + comment atomically
- **Icon**: x-circle (Lucide)
- **Category**: status

Note: 2026-06-28 `Cancelled` status collapsed into `trashedReason`. Single «not-Done» terminal = Trashed; reason captures *why*. For an abandoned (started-not-finished) task, set `trashedReason` accordingly — the command's default reason assumes deliberate drop.
