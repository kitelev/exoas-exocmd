# Visible until terminal (not Done, not Trashed)

Reusable **NotPrecondition** — available ⇔ the wrapped
[[9b85e89a-8162-492f-b85f-2f8a1f21dded|Is terminal status (Done or Trashed)]]
AtomicPrecondition evaluates `false`. I.e. returns `true` while the target
effort's `ems__Effort_status` is anything **except** a terminal status (Done or
Trashed) — the drop-in successor to «Visible until Done» (`37517554`) for
req(ems) `127763f8-b97d-472f-872d-7c21ecdaacf6` (Trashed is a second terminal
status equal to Done).

Replaces the `37517554` conjunct in the active-work gates (the shared composite
`c57642c4`, and — via per-command wrap-the-wrapper — the seven utility
status-gate composites). `37517554` itself is left untouched for «Mark Done Post
Factum», which must stay visible on a Trashed task (the Trashed→Done correction
path).
