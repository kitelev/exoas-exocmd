# $input

Substitution token: user-provided value at command-execution time (typically via modal prompt).

Fail-loud guard: if `targetValue` contains `$input` placeholder but `userInput.value` is undefined/null, grounding execution returns error "property_set: targetValue contains $input/$value placeholder but no userInput.value provided" (RFC-028 Findings 3+4).
