# Fix Missing Label

Generates and sets an `exo__Asset_label` for assets that are missing one, deriving the label from the filename or class name.

- **Precondition**: Visible only when asset has UID but no label (SPARQL ASK)
- **Grounding**: Calls `fixMissingLabel` service
- **Icon**: tag (Lucide)
- **Category**: maintenance
