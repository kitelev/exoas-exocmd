# Is Done and not archived

SPARQL ASK precondition: returns `true` (button shown) when the target asset has `ems__Effort_status` set to `ems__EffortStatusDone` **and** is **not** already archived. Returns `false` (button hidden) once the asset carries the bare `archived: true` frontmatter flag — preventing a re-archive on an already-archived completed asset.

- **Done gate**: `$target ems:Effort_status ems:EffortStatusDone`.
- **Not-archived gate**: `FILTER NOT EXISTS { $target exo:Asset_archived "true" }`. The `archived: true` frontmatter set by the `archiveAsset` service is indexed as the unprefixed-whitelist triple `exo:Asset_archived "true"` (string literal, RFC 871c1e56 A-series `UNPREFIXED_ASSET_FIELDS`).
- Used by the **Archive Completed** command (`52c88678`) to show the button only for completed, not-yet-archived tasks/projects.
