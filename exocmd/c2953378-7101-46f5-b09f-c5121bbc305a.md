# $target

Substitution token: IRI of the current asset (subject of the grounding execution).

Dotted-access pattern: `$target.<property>` resolves to the value of `<property>` on the target asset (e.g. `$target.exo__Asset_label` resolves to the asset's label string).

Most common token (45+ vault uses).
