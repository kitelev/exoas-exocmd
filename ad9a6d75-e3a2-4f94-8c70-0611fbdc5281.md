# exocmd__Grounding_linkBackProperty

`exo__ObjectProperty` on `exocmd__Grounding` (cardinality 0..1).

Указывает имя property на создаваемом ассете (CREATE_INSTANCE flow), в которое будет записан `[[$target]]` wikilink (back-reference на исходный ассет). При отсутствии — fallback на `exo__Asset_source`.

## Disjointness axiom

`owl:propertyDisjointWith exocmd__Grounding_targetProperty` — TBox axiom.

**Rationale (см. RFC `74c5e336-a7a7-47ab-a9f4-3c5abca7b558`):**
- `exocmd__Grounding_targetProperty` — поле на `$target` для операций `property_set` / `property_delete`
- `exocmd__Grounding_linkBackProperty` — поле на **новом** ассете, указывающее обратно на `$target`

Use cases непересекаются → disjoint.
