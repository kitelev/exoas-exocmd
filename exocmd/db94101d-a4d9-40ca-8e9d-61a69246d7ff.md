# exocmd__AtomicPrecondition

Атомарный precondition — лист дерева композиции. Несёт слоты вычисления (`Precondition_sparqlAsk` / `Precondition_query` / `Precondition_hostFunction`); композиты (`AllPrecondition` / `AnyPrecondition` / `NotPrecondition`) ссылаются на него wikilink'ом.
