# Has obsolete properties

Host function precondition: returns `true` when the target asset contains obsolete or deprecated frontmatter properties that should be cleaned up.

Evaluated via `PreconditionEvaluator.registerHostFunction("hasObsoleteProperties", ...)`.
Used by the **Clean Properties** command.
