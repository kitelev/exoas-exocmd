# exocmd__InheritanceRule_priority

`exo__NumberProperty` on `exocmd__InheritanceRule`. Cardinality 0..1, default 50.

YAML serialization:
```yaml

```

Engine sort: rules применяются в порядке `priority desc`. Later writes override earlier for same `_targetProperty`. Используется когда два правила matched для одного `_targetProperty` — higher priority wins.

Default 50 — обычное правило. Higher (100+) — area/condition-specific override. Lower (10) — fallback default.
