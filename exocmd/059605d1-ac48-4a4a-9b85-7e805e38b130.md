# PropertyDefault: exo__Instance_class = $targetClassSelf

`exocmd__PropertyDefault` (project bbe40f8c, T1 «Create Instance»).

Attached to the «Create Instance» Grounding `ef896bf8-116c-4291-8ddb-cc1c744e653b`.
Sets `exo__Instance_class` of the new instance to the `$targetClassSelf`
SubstitutionToken (the host class-definition's own UID). Overrides the Universal
Default Template's `exo__Instance_class = $grounding.targetClass` entry by
property name — the «Create Instance» Grounding declares no `targetClass`
(the host IS the class). uid / createdAt / label still come from the Universal
Default Template.
