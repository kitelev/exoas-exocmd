# Create Instance

`exocmd__Command` for the homoiconic «Create Instance» button (project bbe40f8c,
T1). Rendered on every `exo__Class` definition page (via the binding
`a24e196d-124d-48df-a266-92f1fdcd635f`, `targetClass = exo__Class`).

Click → form (`Label` + `Ontology` fuzzy reference-picker) → a valid UUID
instance of the host class, co-located in the chosen ontology's folder, opened.

- **Grounding**: `Create Instance grounding` (`create_instance`,
  `$targetClassSelf` + `$isDefinedByFolder`).
- **Icon**: circle-plus (Lucide).
- **Category**: creation.
- No confirm dialog, no prefill defaults (per interview requirements).
