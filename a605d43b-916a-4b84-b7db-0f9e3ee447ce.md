# exocmd__Grounding_direction

Datatype property. Domain: `exocmd__Grounding`. Range: xsd:string
(enum: `"forward"` | `"rollback"`). Cardinality: 0..1.

## Semantics

Used only when `exocmd__Grounding_type` is `workflow_transition`. Determines
whether `GroundingExecutor.executeWorkflowTransition` matches transitions
with `isRollback=false` (forward) or `isRollback=true` (rollback). Defaults
to `"forward"` when absent.
