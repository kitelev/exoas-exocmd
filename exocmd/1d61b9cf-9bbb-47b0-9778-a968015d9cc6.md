# Create Instance → ems__ActionPrototype binding

Binds the "Create Action Instance" command (`169fa2ab`) to `ems__ActionPrototype`
assets — sibling of "Create Instance → ems__TaskPrototype binding" (`2d157dc1`).

Task-based "Create Instance"; this restores the manual create path (VL2).
