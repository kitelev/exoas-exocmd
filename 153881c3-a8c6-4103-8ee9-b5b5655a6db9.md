# exocmd__Grounding_targetValueLiteral

`exo__StringProperty` on `exocmd__Grounding` (cardinality 0..1).

Plain string literal value for `property_set` groundings. YAML serialization: `targetValueLiteral: "literal string"`.

## Disjointness axiom

Disjoint with `exocmd__Grounding_targetValueRef` and `exocmd__Grounding_targetValueSubstitution`. TBox cardinality: exactly one of three set per `property_set` grounding (hook-enforced).

Rationale: see RFC `31c1a0be`.
