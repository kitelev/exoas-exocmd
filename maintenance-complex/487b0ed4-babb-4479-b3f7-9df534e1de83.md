# Rollback Status → Task binding

Binds "Rollback Status" command to `ems__Task` assets in the **status** group (was: `maintenance`).

Mirrors v15.38.0 placement where rollback was part of the effort-status workflow alongside Set Status / Mark as Done / Trash. Order=200 places it after the forward-direction Set Status buttons (100..170) within the same group.
