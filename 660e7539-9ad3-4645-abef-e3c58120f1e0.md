# exocmd__SubstitutionToken

Класс substitution-token'ов — placeholder'ов, resolved'имых GroundingExecutor в момент выполнения grounding'а.

## Использование

Используется как range property `exocmd__Grounding_targetValueSubstitution`. Когда grounding указывает `targetValueSubstitution: "[[<token-uid>]]"`, parser смотрит метку токена и подставляет соответствующее значение в `targetValue` перед записью в frontmatter.

Implementation: `packages/exocortex/src/services/GroundingExecutor.ts` — `substituteVariables`.

## Замыкание мира (closed-world)

Class sealed — добавление нового token требует RFC. Все resolver-supported tokens должны иметь corresponding instance в vault.

Source RFC: `31c1a0be-bc72-4a1e-8426-75e7b0ddedd2` (vault-exodev/inbox — RFC: Grounding as first-class RDF).
