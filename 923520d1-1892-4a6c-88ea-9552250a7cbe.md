# Set Status Done

Sets `ems__Effort_status` to Done AND stamps `ems__Effort_endTimestamp` + `ems__Effort_resolutionTimestamp` with the current local timestamp on the target asset.

- **Precondition**: Visible only when asset is NOT in Done status
- **Grounding**: Composite (`77be16d6`) — sets status + endTimestamp + resolutionTimestamp atomically (mirrors v15.38.0 `mark-done` behaviour)
- **Icon**: check-circle (Lucide)
- **Category**: status
