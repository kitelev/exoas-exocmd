# Create Instance → exo__Class binding

`exocmd__CommandBinding` (project bbe40f8c, T1). Binds the «Create Instance»
command to the `exo__Class` **metaclass** (UID
`8619c4fc-64f1-4869-b17e-e34186cacca9`), so the inline button renders on every
class-definition page (every asset whose `exo__Instance_class` includes
`exo__Class`). No precondition — always available on class pages.

`targetClass` is the UID form (not the `exo__Class` label) so the match is
independent of metadata-cache label resolution — the raw UID is always present
in the host's expanded class set (`extractAssetClasses`, Issue #3141).
