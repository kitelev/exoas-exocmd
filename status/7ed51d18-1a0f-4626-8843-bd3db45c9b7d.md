# Allow transition to Blocked (active states only)

SPARQL ASK precondition: returns `true` for any non-terminal status except Blocked itself (avoid no-op transitions).

Used by "Set Status Blocked" command. Blocked is a non-terminal status indicating external dependency wait (vs Waiting which is internal/policy wait).
