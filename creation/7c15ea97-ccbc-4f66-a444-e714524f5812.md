# Create Area → Area binding (child area)

Binds "Create Area" command to `ems__Area` assets so a parent area can spawn a child area directly.

Mirrors v15.38.0 `canCreateChildArea` visibility — Area-only binding enabling hierarchy. Order=280 places it after Create Project (270) and Create Task (260) within the creation group.
