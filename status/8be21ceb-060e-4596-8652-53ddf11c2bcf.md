# Re-open (composite)

Composite grounding for re-opening a Done task/project — reverses the Mark-as-done state:

1. Set `ems__Effort_status` → `[[027e78f4-6e16-4b36-b8fb-5510507d5745]]` (Doing)
2. Delete `ems__Effort_endTimestamp`
3. Delete `ems__Effort_resolutionTimestamp`

Note: `ems__Effort_startTimestamp` is preserved — the original start of work is historically meaningful.

Atomic via `GroundingExecutor.executeComposite` rollback on step failure. Used by command "Re-open".

Replaces the broken universal "Rollback Status" button (Command `2cb208b8`) whose `rollbackStatus` service was broken by UUID-canon TBox refactor — `WorkflowEngine.getPreviousStatus` could not match UUID-form status against symbolic-form transitions.
