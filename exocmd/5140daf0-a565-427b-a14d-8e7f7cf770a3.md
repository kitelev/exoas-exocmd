# Not in This Week zone

SPARQL ASK precondition: returns `true` when task zone is NOT This Week AND task is not in a terminal status (Done / Cancelled / Trashed / Failed). Zone uses filename-IRI form (human-labelled shared-identities asset), status uses symbolic IRI form (`ems__EffortStatus*` labels).

Used by "Set Criticality Medium" command to show the button only when relevant.
