# Not in Someday zone — zone/status atomic

**AtomicPrecondition** — the zone+status remainder of the original "Not in Someday zone" precondition (`10c1d396`), with the shared `NOT EXISTS { … Prototype }` guard factored out. Returns `true` when task zone is NOT Someday AND task is not in a terminal status (Done / Cancelled / Trashed / Failed).

Composed with the shared not-prototype guard via AllPrecondition in the re-typed `10c1d396` composite. **Behavior-preserving split** — copied verbatim from the original ASK minus the prototype fragment. Zone target uses filename-IRI form, status uses symbolic IRI (copied verbatim from `10c1d396`).
