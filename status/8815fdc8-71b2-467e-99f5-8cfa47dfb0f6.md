# Allow transition to Backlog (from Draft / Cancelled)

SPARQL ASK precondition: returns `true` only when current status ∈ {Draft, Cancelled}.

State-machine forward transition: Draft → Backlog is the standard onboarding path; Cancelled → Backlog allows resurrecting a cancelled asset back into the workflow. From any other status, use Rollback Status (one step back).

Used by "Set Status Backlog" command.
