# Set startTimestamp to now

Grounding step: writes current local timestamp to `ems__Effort_startTimestamp`.

`$nowLocal` is expanded by `GroundingExecutor.substituteVariables` to the same format produced by `TaskStatusService.startEffort` (v15.38.0 reference behaviour).

Designed as a step inside composite groundings — chains with `Set status to Doing` to atomically mirror v15.38.0 `start-effort` command semantics.
