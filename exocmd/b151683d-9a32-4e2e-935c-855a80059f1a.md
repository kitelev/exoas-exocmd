# exocmd__Template_propertyDefault

`exo__ObjectProperty` on `exocmd__UniversalDefaultTemplate`. Multi-valued list of `exocmd__PropertyDefault` instance wikilinks. Cardinality 1..N.

YAML serialization:
```yaml
exocmd__Template_propertyDefault:
  - "[[<property-default-uuid-1>]]"
  - "[[<property-default-uuid-2>]]"
```

Engine merges these with Grounding's own `Grounding_propertyDefault` list at parse-time in `CommandResolver.resolvePropertyDefaults` (RFC 727572d2 Phase B). Grounding entries with matching `propertyName` override Universal entries.
