# Target is not a prototype

SPARQL ASK precondition: returns `true` when the target asset is **not** a
prototype-template — i.e. none of its `exo__Instance_class` values reaches
`exo__Prototype` (UID `ebf717aa-4070-4b37-abde-10a700e354fc`) through the
transitive `exo:Class_superClass*` chain.

Because every `XPrototype` class (e.g. `ems__TaskPrototype`) holds `exo__Prototype`
in its superClass closure, this is `false` for any prototype-template asset and
`true` for an ordinary instance. Used to hide instance-lifecycle commands
(Start / Mark Done / Set Criticality / status transitions / child-creation /
convert / vote) from prototype-template assets while leaving
universal-maintenance (`targetClass exo__Asset`) and the prototype's own
create-instance command untouched.

Realises requirement `5579ffa1-a829-4fcf-b93d-4fb664b02d62` (design node
`f853eadd-6132-43f8-84ba-25a75641096a`). The same `FILTER NOT EXISTS { … }`
clause is conjoined into the 17 existing lifecycle status/criticality/convert
preconditions so the not-a-prototype gate ANDs with their status gates.
