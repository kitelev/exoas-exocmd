# Execute

One-tap "Execute" command for atomic `ems__Action` assets (epic `40188c71`).
Sets `ems__Effort_status` → Done (`7b9b3116`) **and** stamps
`ems__Effort_resolutionTimestamp` (`6c5ea4a6`) with the completion moment — via the
composite grounding `638751e3` ("Execute action").

Unlike the Task "Mark Done" command (`923520d1`, which requires status = Doing
first), this uses the permissive precondition `37517554` ("Visible until Done") so
an atomic action transitions **any-status → Done in a single tap** — e.g.
medication intake (Draft/Backlog → Done). The precondition hides the button once
the action is Done.

(Was previously labelled "Complete"; renamed to "Execute" per the epic vision,
2026-06-29.)
