# Has non-UID filename

SPARQL ASK precondition: returns `true` when the target asset's filename does not match its `exo__Asset_uid` (i.e., is a human-readable name instead of UUID-based).

Reads `exo:Asset_filename` (parser-derived basename triple, materialized since `@kitelev/exocortex-cli@16.16.0`, RFC 1ce2a226 Phase 3) and compares lexically against `exo:Asset_uid`.

Used by the **Rename to UID** command (legacy precondition kept for direct reference; `hasNonUidFilenameOutsideWhitelist` is the whitelist-aware variant wired into the command, RFC Phase 5).
