# Allow transition to Cancelled (any non-terminal)

SPARQL ASK precondition: returns `true` for any current status except Done and Cancelled (terminal states).

State-machine kill: mirrors v15.38.0 `canTrashEffort` — abandon work from any active state. Done assets can't be re-cancelled (use Rollback Status if needed). The grounding is composite (status + resolutionTimestamp).

Used by "Set Status Cancelled" command.
