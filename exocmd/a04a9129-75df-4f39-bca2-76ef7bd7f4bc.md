# Copy Label to Aliases

Appends the `exo__Asset_label` value to the asset's `aliases` list via the declarative `property_append` grounding — no TypeScript service code is involved (Homoiconicity Invariant Q1 remediation, Issue #3132).

- **Grounding**: declarative `property_append` (`appendExpression: $target.exo__Asset_label`, Set-based dedup → idempotent)
- **Precondition**: [[888c34ba-75e9-4fbf-b430-5ab5dbed8f4a|Label not yet in aliases]] — homoiconic SPARQL ASK (`label ∉ exo:Asset_aliases`); hides the button once the label is already an alias (RFC 1b1c21b2 D3)
- **Icon**: copy (Lucide)
- **Category**: misc
