# exocmd__Grounding_targetValueRef

`exo__ObjectProperty` on `exocmd__Grounding` (cardinality 0..1).

RDF-reference value for `property_set` groundings. YAML serialization: `targetValueRef: "[[<asset-uid>]]"`.

## Disjointness axiom

Disjoint with `exocmd__Grounding_targetValueLiteral` and `exocmd__Grounding_targetValueSubstitution`. TBox cardinality: exactly one of three set per `property_set` grounding (hook-enforced).

Rationale (RFC `31c1a0be`):
- Explicit semantics over union-type targetValue (parser dispatch zero heuristics)
- Trivial SPARQL: `?g exocmd:Grounding_targetValueRef ?ref` — direct typed query
- Aligns with `linkBackProperty` disjointness precedent (RFC `74c5e336`)
