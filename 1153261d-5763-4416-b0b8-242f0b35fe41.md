# Set reviewTimestamp to now

Grounding step: writes current local timestamp to `ems__Effort_reviewTimestamp`.

`targetProperty` is authored in **symbolic-string form** (`"ems__Effort_reviewTimestamp"`) matching the existing precedent (`bdf3c81f` "Set status to Done"). This is the format the current pre-Phase-3 parser correctly resolves via `CommandResolver.getObsidianName` → `unwrapWikilink` (returns the string unchanged).

**Earlier wikilink-form `[[65f3f5d9-...]]`** was reverted (self-review 2026-05-19 HIGH-4): the parser path for that form returns the bare UUID (`unwrapWikilink` strips brackets) which would then be written as a UUID-named frontmatter key — data corruption. Phase 3 parser PR fixes that path; until then symbolic-string keeps the grounding safe to invoke today.

`$nowLocal` substitution token matches the format produced by `StatusTimestampService.addReviewTimestamp` (called by legacy `TaskStatusService.markAsReviewed`, to be deleted in Phase 4).

Used by vault Command "Mark Reviewed" (`e42f2e7c-dfab-47b9-8e7e-2edfd7a08d95`) — single-step property_set grounding. Bound to `ems__Task` via CommandBinding `0b07b6f5`.
