# Has empty properties

Host function precondition: returns `true` when the target asset contains at
least one **empty** frontmatter property (a value that is `null`, an empty
string, an empty array, or an empty object) — exactly the properties the
`cleanProperties` service removes.

Evaluated via `PreconditionEvaluator.registerHostFunction("hasObsoleteProperties", …)`
(the `hasObsoleteProperties` name is a legacy identifier; the registered factory
reads the target's frontmatter and returns the core `hasEmptyProperties` helper).
Used by the **Clean Properties** command.
