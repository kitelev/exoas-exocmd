# Execute action (composite)

Composite grounding for the "Execute" command (`f86925c4`) on `ems__Action` assets.
Mirrors the epic `40188c71` Execute spec exactly — two atomic property_set steps:

1. Set `ems__Effort_status` → Done (`7b9b3116`) — step `bdf3c81f` (reused).
2. Set `ems__Effort_resolutionTimestamp` (`6c5ea4a6`) → `$nowLocal` — step `ee5c36a3` (reused).

Unlike the Task "Mark as done (composite)" (`77be16d6`), this omits the
`ems__Effort_endTimestamp` step — an atomic Action has no duration, only a
resolution moment (`resolvedAt`). Atomic via `GroundingExecutor.executeComposite`
(rollback on step failure).
