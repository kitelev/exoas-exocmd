# Not in Today zone — zone/status atomic

**AtomicPrecondition** — the zone+status remainder of the original "Not in Today zone" precondition (`00ffed4e`), with the shared `NOT EXISTS { … Prototype }` guard factored out. Returns `true` when task zone is NOT Today AND task is not in a terminal status (Done / Cancelled / Trashed / Failed).

Composed with the shared not-prototype guard via AllPrecondition in the re-typed `00ffed4e` composite. **Behavior-preserving split** — copied verbatim from the original ASK minus the prototype fragment.

**IRI forms** — zone target uses the *filename-IRI* form (`obsidian://vault/.../<uuid>.md`) because the zone enum asset carries a human-language label; status enums use the symbolic IRI (`ems__EffortStatus*`). Copied verbatim from `00ffed4e`.
