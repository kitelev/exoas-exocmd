# Is Prototype

SPARQL ASK precondition that checks whether the target asset is an instance of `exo__Prototype` (directly or via subclass hierarchy).

- Matches assets with direct `exo:Instance_class exo:Prototype` (e.g. `ztlk__FleetingNotePrototype`)
- Also matches assets whose class inherits from `exo__Prototype` via `exo:Class_superClass+`
- Used by "Create Instance" command bindings to show the button only on prototype assets
