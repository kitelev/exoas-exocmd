# exocmd__PropertyDefault

Declarative default-value rule attached to `exocmd__Grounding` via `Grounding_propertyDefault`.

## Properties (Phase 1)

- `exocmd__PropertyDefault_property` — какое property pre-fill (range: `exo__Property`)
- `exocmd__PropertyDefault_value` — какое значение записать (range: `exo__Asset` — wikilink-form, possibly to `SubstitutionToken` instance for dynamic resolution)

## Engine semantics

`GroundingExecutor.executeCreateInstance` shouldевалюировать каждый PropertyDefault and fill `frontmatter[property.label] = resolvedValue` ДО copy-from-target step (см. RFC §Precedence).

Idempotency key (Phase 2 migration): unique pair `(Grounding-uid, PropertyDefault_property-uid)`.
