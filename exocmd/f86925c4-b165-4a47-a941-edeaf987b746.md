# Complete

One-tap completion command for atomic `ems__Action` assets. Sets `ems__Effort_status` → Done **and** stamps `ems__Effort_endTimestamp` + `ems__Effort_resolutionTimestamp` with the actual completion time (composite grounding `77be16d6`, reused from "Mark Done").

Unlike the Task "Mark Done" command (`923520d1`, which requires status = Doing first), this uses the permissive precondition `37517554` (visible until Done) so an atomic action transitions **any-status → Done in a single tap** — e.g. medication intake (Backlog → Done). The precondition hides the button once the action is Done.
