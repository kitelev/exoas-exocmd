# Rename to UID via service

Grounding action that invokes the `renameToUid` registered service to rename the target asset's file to match its `exo__Asset_uid` value (e.g., `My Task.md` -> `a1b2c3d4-...md`).

Uses `service_call` grounding type with `serviceId: renameToUid`.
