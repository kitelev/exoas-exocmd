# Set resolutionTimestamp to now

Grounding step: writes current local timestamp to `ems__Effort_resolutionTimestamp`.

`$nowLocal` matches the format produced by `TaskStatusService.markTaskAsDone` and `TaskStatusService.trashEffort` (v15.38.0 reference behaviour).

Reused by composite groundings for "Set Status Done" and "Set Status Cancelled" — both terminal-state transitions stamp the resolution moment.
