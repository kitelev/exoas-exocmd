# Create kitelev__SelfObservation instance (Чувствую усталость)

Grounding for the `Чувствую усталость` prototype → creates a `kitelev__SelfObservation` instance on click.

- `Grounding_type = create_instance` → handled by `GroundingExecutor.executeCreateInstance` (`packages/exocortex/src/services/GroundingExecutor.ts`).
- `targetClass = kitelev__SelfObservation` → new file gets `exo__Instance_class: [[c862dfda-85d2-4dc0-892e-f0efdcd054f4|kitelev__SelfObservation]]`.
- `targetPrototype = "Чувствую усталость"` → new file gets `exo__Asset_prototype: [[1c1b0287-d194-491d-b893-4e721aa0444b|Чувствую усталость]]`.
- `targetFolder = 03 Knowledge/kitelev` → new instance created next to its prototype.
- `inputSchema` provides a single `label` field, defaulting to «Чувствую усталость» (user can override per-event).
- `prefillLabelWithDate: true` → modal will pre-pad label with current date for distinguishability.

Sibling of Create Task/Project/FleetingNote grounding variants (see [[bb00efed-7b17-42f5-a2c4-7cadf3e0ab36]]).
