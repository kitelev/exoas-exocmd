# exocmd__DeclarativeRule (abstract)

Abstract superclass для attachable declarative rules в exocmd ontology.

## Purpose

Pure taxonomy node без properties. Enables generic SPARQL queries «all declarative rules attached to entity X» without enumerating subclasses.

## Grounding-attached subclasses (Phase 1)

- `exocmd__PropertyDefault` — default-value declaration для property_set / create_instance Groundings
- `exocmd__InheritanceRule` — target-property mapping rule (source → target with optional class condition / exclusion)
- `exocmd__SubstitutionToken` — named substitution variable (`$today`, `$todayStart`, …)

## UI-attached subclasses (Phase 5 retrofit)

- `exocmd__CommandBinding` — UI-attachment rule (Class → Command)
- `exocmd__Precondition` — visibility predicate rule (SPARQL ASK)
