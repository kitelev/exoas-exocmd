# Execute

One-tap "Execute" command for atomic `ems__Action` assets (epic `40188c71`).
Sets `ems__Effort_status` → Done (`7b9b3116`) **and** stamps
`ems__Effort_resolutionTimestamp` (`6c5ea4a6`) with the completion moment — via the
composite grounding `638751e3` ("Execute action").

Unlike the Task "Mark Done" command (`923520d1`, which requires status = Doing
first), this uses the composite precondition `c57642c4` = `AllPrecondition[37517554
"Visible until Done", 5b49ef19 "Not a prototype"]` so an atomic action transitions
**any-status → Done in a single tap** — e.g. medication intake (Draft/Backlog →
Done). The precondition hides the button once the action is Done **and** on
`ems__ActionPrototype` templates (you execute instances, not the template — D3
curation, 2026-07-05).

2026-06-29.)
