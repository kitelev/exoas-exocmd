# Allow transition to Done (from Doing)

SPARQL ASK precondition: returns `true` only when current status = Doing.

State-machine terminal forward transition mirroring v15.38.0 `canMarkDone`. Doing → Done is the standard completion. Done is terminal — the only way out is Rollback Status (back to Doing).

Used by "Set Status Done" command. The grounding is composite (status + endTimestamp + resolutionTimestamp).
