# Allow transition to Trashed (any non-terminal)

SPARQL ASK precondition: returns `true` for any current status except Done and Trashed (terminal states).

State-machine kill: mirrors `canTrashEffort` — abandon/drop work from any active state. Done/Trashed assets can't be re-trashed (use Rollback Status if needed). The grounding is composite (status + reason + resolutionTimestamp + comment).

Used by the "Trash" command (`0c3debe0`).
