# $userInput

`exocmd__SubstitutionToken` instance. **Parameterised.** Resolves to `userInput[<parameter>]` where `<parameter>` is the modal-collected input key.

## Parameter

String literal: the userInput map key to read. Example parameter values:
- `description` → reads `userInput.description`
- `priority` → reads `userInput.priority`

## Invocation shape

This token must be referenced via a TokenInvocation wrapper (since it requires a parameter). The wrapper instance frontmatter sets `exocmd__TokenInvocation_token` to this token's UID and `exocmd__TokenInvocation_parameter` to the userInput key string. Then `PropertyDefault_value` references the TokenInvocation wrapper.
