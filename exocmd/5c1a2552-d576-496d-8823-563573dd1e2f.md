# exocmd__GroundingTypeWorkflowTransition

Grounding catalog instance for the `workflow_transition` dispatch path
(RFC 36347daf Phase 2).

Differs from other grounding types because it is **runtime-resolved**:
the actual mutation depends on (assetClass, currentStatus, direction)
lookups against vault Workflow ABox, not a fixed `targetProperty`.

## Companion property

`exocmd__Grounding_direction` — `"forward"` | `"rollback"`. Defaults to
`"forward"` when omitted.
