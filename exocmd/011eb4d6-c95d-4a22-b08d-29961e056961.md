# Is prototype

Reusable **AtomicPrecondition** — returns `true` when the target's `exo__Instance_class` resolves to a class whose IRI ends with `Prototype` (i.e. the target IS a prototype). Extracted from the duplicated inline `NOT EXISTS { … STRENDS "Prototype" }` guard copied across ~20 preconditions (RFC df602adc Phase 6).

Wrapped by a NotPrecondition (`5b49ef19-7eb9-4b67-bb4e-c9002537000b`, "Not a prototype") to get the negation — the "not-prototype" guard used by the criticality-zone commands. Composing this single leaf into multiple composites makes reuse / CQ3 non-vacuous.
