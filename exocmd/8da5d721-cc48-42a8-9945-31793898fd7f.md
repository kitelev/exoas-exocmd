# $targetClassSelf

`exocmd__SubstitutionToken` instance (project bbe40f8c, T1 «Create Instance»).

Resolves to a quoted wikilink to the **click-target file's own UID** —
`"[[<hostUid>]]"`. Used for `exo__Instance_class` of a new instance created from
a class-definition page: the host page IS the class, so the instance's class is
the host. Inverts `$grounding.targetClass` (which reads a class baked into the
Grounding). Context-dependent → resolved at execution time by
`GroundingExecutor` via `SubstitutionResolverRegistry`.
