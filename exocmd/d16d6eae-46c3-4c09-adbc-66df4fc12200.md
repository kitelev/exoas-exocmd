# $target.property

`exocmd__SubstitutionToken` instance. **Parameterised.** Resolves to the value of `targetFm[<parameter>]` — reads a property from the click target asset's frontmatter.

## Parameter

String literal: the property label whose value to extract. Examples:
- `exo__Asset_isDefinedBy` → reads target's isDefinedBy field
- `ems__Effort_area` → reads target's area field

## Invocation shape

Always referenced via a TokenInvocation wrapper. RFC 727572d2 Universal Default uses this to inherit `exo__Asset_isDefinedBy` from the target onto every newly created instance.
