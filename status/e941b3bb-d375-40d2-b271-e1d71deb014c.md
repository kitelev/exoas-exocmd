Sets `ems__Effort_status` to Doing AND stamps `ems__Effort_startTimestamp` with the current local timestamp on the target asset.

- **Precondition**: Visible only when asset is NOT in Doing status
- **Grounding**: Composite (`abae3a07`) — sets status + startTimestamp atomically (mirrors v15.38.0 `start-effort` behaviour)
- **Icon**: play-circle (Lucide)
- **Category**: status
