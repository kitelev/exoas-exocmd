# Archive Completed

Moves a completed (Done status) asset to the archive folder and marks it as archived.

- **Precondition**: Visible only when asset status is Done **and not yet archived** (SPARQL ASK `8762ddc2` "Is Done and not archived")
- **Grounding**: Calls `archiveAsset` service
- **Icon**: archive (Lucide)
- **Category**: maintenance
