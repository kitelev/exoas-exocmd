# Cancel effort (composite)

Composite grounding mirroring v15.38.0 `trash-effort` command behaviour (terminal-state with resolution timestamp):

1. Set `ems__Effort_status` → `[[cf6269e4-6c5e-418e-b9f2-115aea5fcfb6]]`
2. Set `ems__Effort_resolutionTimestamp` → current local timestamp

Atomic via `GroundingExecutor.executeComposite` rollback on step failure. Used by command `Set Status Cancelled` (`0c3debe0`).

Note: v15.38.0 used `EffortStatusTrashed` (Trash command); current vault model uses `EffortStatusCancelled` as the equivalent terminal state. Resolution-timestamp semantics preserved.
