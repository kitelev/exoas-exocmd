# Is in wrong folder

Host function precondition: returns `true` when the target asset's file is located in an incorrect folder according to the ontology-driven folder rules (class-to-folder mapping).

Evaluated via `PreconditionEvaluator.registerHostFunction("isInWrongFolder", ...)`.
Uses `EvalContext.currentFolder` to check current location against expected folder.
Used by the **Repair Folder** command.
